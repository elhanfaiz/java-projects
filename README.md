# Java Projects Collection

This repository contains a collection of beginner to intermediate Java console-based projects created to practice core programming concepts, object-oriented programming (OOP), and problem-solving skills.

---

## Projects Included

### Guess Number Game
A simple number guessing game where the user tries to guess a randomly generated number.

**Concepts Used:**
- Loops
- Conditions
- Random class
- User input

---

### Calculator
A basic calculator that performs arithmetic operations like addition, subtraction, multiplication, and division.

**Concepts Used:**
- Switch statement
- Methods
- User input

---

### Student Average Calculator
Calculates the average marks of students and determines grades.

**Concepts Used:**
- Arrays
- Loops
- Conditional statements

---

### To-Do List
A console-based task manager to add and display tasks.

**Concepts Used:**
- ArrayList
- Loops
- Menu-driven program

---

### Shopping List
Stores and displays shopping items entered by the user.

**Concepts Used:**
- Arrays 
- Loops

---

### Banking Management System
Simulates a simple banking system with deposit, withdraw, and balance checking features.

**Concepts Used:**
- OOP
- Methods
- Conditional logic

---

### Library Management System
Allows adding books, issuing books, returning books, and viewing available books.

**Concepts Used:**
- Classes & Objects
- ArrayList
- Boolean flags
- Menu-driven system

---

### ATM System with PIN
Simulates an ATM machine with PIN authentication, deposit, withdrawal, and balance checking.

**Features:**
- 3 PIN attempts
- Secure login
- Transaction handling

**Concepts Used:**
- Encapsulation
- OOP
- Loops & conditions

---

### Employee Management System
Manages employee records including adding, viewing, searching, updating salary, and deleting employees.

**Concepts Used:**
- OOP
- ArrayList
- CRUD operations
- Menu-driven logic

---

## Technologies Used

- Java
- OOP Concepts
- ArrayList
- Loops & Conditions
- Console-based input/output

---

## How to Run

1. Compile the file:
   javac FileName.java

2. Run the program:
   java FileName
